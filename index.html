Frontend Simplified

Frontend Simplified HTML training course is good for people who want to start learning coding. First we learn that Element names are surrounded by angle brackets. They usually come in pairs but some close themselves.

H tags and P tags are block tags, they start on a new line and take the full width available. Some Tags such as the bold tag can be used inside other elements as they are inline level tags. Inline level tags dont start on a new line and only take up the necessary space. The Div tag is used to group elements together and has no actual meaning. Italices is another inline level tag.
